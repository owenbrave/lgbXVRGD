# 项目简介

## 核心功能
- 基于YOLOv8的暴力行为检测，包含4类识别：非暴力、暴力、枪、刀
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 实时显示检测结果：目标位置、目标总数、置信度等信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 提供模型选择功能
- 批量检测并查看所有结果
- 支持检测结果保存

## 适用场景
- 公共场所安全监控
- 视频内容安全审核
- 暴力行为研究分析
- 安防系统开发
- 计算机视觉教学演示

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于python3.9，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供完整源码、数据集(13600张)、训练好的模型及结果
- 包含UI界面源文件、图标和详细运行教程
- 已配置成功环境，可直接使用
